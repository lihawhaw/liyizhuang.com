前置依赖

- **iTerm2** 安装 https://iterm2.com
- **zsh** 安装 brew install zsh
- **Oh My Zsh** 安装 https://ohmyz.sh
- **字体** 安装 [MesloLGS NF](https://github.com/romkatv/powerlevel10k?tab=readme-ov-file#meslo-nerd-font-patched-for-powerlevel10k)

```shell
Do all these icons fit between the crosses? (y)
Prompt Style (1)
Character Set (1)
Prompt Colors (1)
Show current time? (2)

Prompt Spacing (1)
Icons (2)

```
