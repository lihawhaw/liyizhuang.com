这个网站从 2019.9.2 号开始搭建到现在(2023.4.8)，也有三年多的时间了，从刚开始的 [Hexo](https://hexo.io/)，到后来的 [React](https://react.dev) + [antd](https://ant.design/)， 一直都是把这个域名当来做 api 接口域名在使用，基本很少会打开网站，去年开始把接口陆陆续续切换到了 [lihaha.cn](https://lihaha.cn) 域名上面，这个网站打开的次数就更少了，前段时间在给其他域名更新 SSL 证书的时候，打开这个网站发现首次加载并没有想象的那么快了，虽然 cdn 加速还在，但是一些图片资源什么的已经打不开了，基本上就是年久失修，之前做的一些图片库什么的，早就被我给下掉了，导致这个网站看起来破旧不堪，就想着怎么重构下，但是因为我基本上很少会在这里更新文章，所以就想怎么用最小的成本重构一下，再体验下新的前端框架什么的，最好是 ssg 类型的，这两天看到了 [astro](https://astro.build/) ，不仅轻量，还支持介入任何语言的组件，当时就感觉这就是我想要的，然后趁着这次周末开始重构。
